Pemrograman Web: Materi & Contoh Kode

Pertemuan 1: HTML Dasar

HTML digunakan untuk membuat struktur dasar halaman web.

<!DOCTYPE html>
<html>
<head>
    <title>Halo Dunia</title>
</head>
<body>
    <h1>Selamat Datang di Pemrograman Web</h1>
</body>
</html>

Pertemuan 2: CSS Dasar

CSS digunakan untuk mengatur tampilan halaman web.

body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
}
h1 {
    color: navy;
    text-align: center;
}

Pertemuan 3: JavaScript Dasar

JavaScript digunakan untuk menambahkan interaktivitas pada halaman web.

function sapa() {
    alert("Halo, Selamat Belajar JavaScript!");
}

Pertemuan 4: Pengenalan PHP

PHP adalah bahasa pemrograman sisi server untuk membuat halaman dinamis.

<?php
    echo "Halo, ini adalah PHP!";
?>

Pertemuan 5: Mengontrol Alur Program

Struktur kontrol seperti if-else digunakan untuk membuat keputusan dalam kode.

<?php
$nilai = 80;
if ($nilai >= 75) {
    echo "Anda Lulus!";
} else {
    echo "Anda Tidak Lulus!";
}
?>

Pertemuan 6: Perulangan dan Array

Array digunakan untuk menyimpan banyak data, dan perulangan digunakan untuk mengaksesnya.

<?php
$buah = ["Apel", "Jeruk", "Mangga"];
foreach ($buah as $b) {
    echo $b . "
";
}
?>
